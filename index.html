HTML

  • The HTML element is the root element and encloses the entire document
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML tags are used to define the structure of the document using headings, paragraphs, lists and links
  • Tags are enclosed in angle brackets and have an opening and clsoing tag
  • All HTML have a document type declaration (DTD) at the beginning which tells the browser which version of HTML

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • Selectors are used to target specific elements such as 'p' or '.class' for elements with a specific class
  • Declarations consists of property and a value, separated by a colon. Must have a selector
  • CSS can be included in HTML using a 'style' tag in head or an external CSS file that is linked using rel and href

Git

  • git staus: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches it
  • feature branch: branch separate from main branch that adds new feature

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • It can be used to implement conditional statements, loops, and functions
  • It includes built-in objects such as Math, Date, and String objects
